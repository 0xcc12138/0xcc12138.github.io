Windows x64内核

天堂之门简介

在x86-64 架构中，有一个特殊的机制可以让 CPU 在 64 位模式和 32 位模式之间进行切换。Heaven's Gate 就是利用这个机制，在 32 位环境下进入 64 位模式，从而执行 64 位代码并调用 64 位 API。

在长模式下（即 64 位处理器启用 64 位功能时），段选择子可以用来确定 CPU 运行在 64 位模式还是兼容模式。 （具体是CS段寄存器的L位，设置为1就是64位模式）
