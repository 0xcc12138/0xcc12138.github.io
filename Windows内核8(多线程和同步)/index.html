IRQLпјҲInterrupt Request LevelпјҢдёӯж–ӯиҜ·жұӮзә§еҲ«пјүжҳҜWindowsж“ҚдҪңзі»з»ҹеҶ…ж ёдёӯзҡ„дёҖдёӘе…ій”®жҰӮеҝөпјҢз”ЁдәҺз®ЎзҗҶе’ҢжҺ§еҲ¶дёӯж–ӯе’ҢзәҝзЁӢи°ғеәҰгҖӮIRQLеҶіе®ҡдәҶеҪ“еүҚд»Јз Ғзҡ„жү§иЎҢдјҳе…Ҳзә§пјҢеҪұе“ҚдәҶе“Әдәӣдёӯж–ӯе’ҢзәҝзЁӢеҸҜд»Ҙиў«и°ғеәҰе’ҢеӨ„зҗҶгҖӮ

е®ҡд№үпјҡIRQLжҳҜдёҖдёӘж•ҙж•°еҖјпјҢиЎЁзӨәеҪ“еүҚеӨ„зҗҶеҷЁеңЁзү№е®ҡж—¶й—ҙзӮ№зҡ„дёӯж–ӯиҜ·жұӮзә§еҲ«гҖӮ

дҪңз”ЁпјҡIRQLз”ЁдәҺжҺ§еҲ¶дёӯж–ӯзҡ„еӨ„зҗҶйЎәеәҸе’ҢзәҝзЁӢзҡ„и°ғеәҰпјҢзЎ®дҝқй«ҳдјҳе…Ҳзә§зҡ„дёӯж–ӯиғҪеӨҹеҸҠж—¶еӨ„зҗҶпјҢйҳІжӯўдҪҺдјҳе…Ҳзә§зҡ„д»»еҠЎе№Іжү°й«ҳдјҳе…Ҳзә§д»»еҠЎзҡ„жү§иЎҢгҖӮ

3. IRQLзҡ„дҪңз”Ё

  • дёӯж–ӯеӨ„зҗҶпјҡIRQLжҺ§еҲ¶дёӯж–ӯзҡ„дјҳе…Ҳзә§пјҢзЎ®дҝқй«ҳдјҳе…Ҳзә§зҡ„дёӯж–ӯиғҪеӨҹжҠўеҚ дҪҺдјҳе…Ҳзә§зҡ„дёӯж–ӯеӨ„зҗҶгҖӮдҫӢеҰӮпјҢзЎ¬зӣҳдёӯж–ӯйҖҡеёёе…·жңүй«ҳдјҳе…Ҳзә§пјҢд»ҘзЎ®дҝқж•°жҚ®зҡ„еҝ«йҖҹиҜ»еҶҷгҖӮ
  • зәҝзЁӢи°ғеәҰпјҡж“ҚдҪңзі»з»ҹж №жҚ®зәҝзЁӢзҡ„дјҳе…Ҳзә§е’ҢеҪ“еүҚIRQLжқҘеҶіе®ҡзәҝзЁӢзҡ„и°ғеәҰйЎәеәҸгҖӮй«ҳдјҳе…Ҳзә§зҡ„зәҝзЁӢеҸҜд»ҘеңЁй«ҳIRQLдёӢиҝҗиЎҢпјҢд»ҘзЎ®дҝқе…ій”®д»»еҠЎзҡ„еҸҠж—¶е®ҢжҲҗгҖӮ
  • еҗҢжӯҘжңәеҲ¶пјҡIRQLз”ЁдәҺе®һзҺ°еҶ…ж ёеҗҢжӯҘжңәеҲ¶пјҢдҫӢеҰӮиҮӘж—Ӣй”ҒгҖӮжҸҗй«ҳIRQLеҸҜд»ҘйҳІжӯўеҪ“еүҚд»Јз Ғиў«дёӯж–ӯпјҢд»ҺиҖҢдҝқжҠӨдёҙз•ҢеҢәгҖӮ

4. IRQLзҡ„з®ЎзҗҶ

  • жҸҗеҚҮIRQLпјҡеңЁжҹҗдәӣжғ…еҶөдёӢпјҢеҶ…ж ёд»Јз ҒйңҖиҰҒжҸҗеҚҮIRQLд»ҘйҳІжӯўиў«дёӯж–ӯгҖӮдҫӢеҰӮпјҢеңЁеӨ„зҗҶе…ій”®ж•°жҚ®з»“жһ„ж—¶пјҢд»Јз ҒеҸҜиғҪдјҡе°ҶIRQLжҸҗеҚҮеҲ°DISPATCH_LEVELд»ҘйҳІжӯўдёҠдёӢж–ҮеҲҮжҚўгҖӮ
  • йҷҚдҪҺIRQLпјҡд»Јз ҒеңЁе®ҢжҲҗе…ій”®д»»еҠЎеҗҺйңҖиҰҒжҒўеӨҚд№ӢеүҚзҡ„IRQLпјҢд»Ҙе…Ғи®ёе…¶д»–дёӯж–ӯе’ҢзәҝзЁӢжү§иЎҢгҖӮ
  • IRQLжЈҖжҹҘпјҡеҶ…ж ёжҸҗдҫӣдәҶдёҖдәӣAPIз”ЁдәҺжЈҖжҹҘе’Ңж“ҚдҪңеҪ“еүҚзҡ„IRQLпјҢеҰӮKeGetCurrentIrqlгҖҒKeRaiseIrqlе’ҢKeLowerIrqlгҖӮ

еҫ®иҪҜи®ҫи®ЎдәҶ 0~31зә§еҲ«пјҢеңЁзЁӢеәҸеҶ…пјҢеҸҜд»Ҙз”Ёд»Јз ҒжҠҠиҮӘе·ұзҡ„зә§еҲ«жҸҗй«ҳжҲ–иҖ…йҷҚдҪҺгҖӮйӮЈд№ҲжңүжІЎжңүеҸҜиғҪиҪҜд»¶зҡ„зә§еҲ«жҜ”зЎ¬д»¶й«ҳпјҹ

еҫ®иҪҜи®ҫи®ЎдәҶиҪҜд»¶еҸӘиғҪжҳҜ02зә§пјҢиҖҢзЎ¬д»¶жңү331зә§гҖӮиҝҷж ·еҚідҪҝиҪҜд»¶жқғйҷҗжҸҗеҲ°жңҖй«ҳпјҢд№ҹдёҚиҮідәҺеҪұе“ҚеҲ°зЎ¬д»¶ж“ҚдҪң

дҫӢеҰӮзЎ¬д»¶дёҠзҡ„з”өжәҗзә§еҲ«е°ұжҳҜPOWER_LEVEL 30пјҢеҮ д№ҺжҳҜжңҖй«ҳзҡ„дәҶпјҢйӮЈд№Ҳд»ЈиЎЁиҪҜд»¶еңЁиҝҗиЎҢзҡ„ж—¶еҖҷпјҢеҰӮжһңжҢүдёӢдәҶз”өжәҗй”®пјҢдёҚдјҡз®Ўд»Јз ҒжңүжІЎжңүиҝҗиЎҢе®ҢпјҢе°ұжү§иЎҢе…іжңәзҡ„жҢҮд»ӨдәҶ

иҪҜд»¶зҡ„дёүдёӘзә§еҲ«пјҢеҲҶеҲ«жҳҜ

еңЁDISPATCH_LEVELиҝҗиЎҢзҡ„д»Јз ҒдёҚиғҪеӨ„зҗҶPASSIVE_LEVELзҡ„ејӮеёёгҖӮDISPATCH_LEVELжҳҜдёҖдёӘиҫғй«ҳзҡ„дёӯж–ӯиҜ·жұӮзә§еҲ«пјҢз”ЁдәҺи°ғеәҰе’ҢеӨ„зҗҶй«ҳдјҳе…Ҳзә§зҡ„д»»еҠЎгҖӮжӯӨж—¶пјҢи®ёеӨҡзі»з»ҹеҠҹиғҪжҳҜеҸ—йҷҗзҡ„пјҢжҜ”еҰӮеҶ…еӯҳеҲҶйЎөпјҢж“ҚдҪңж–Үд»¶е’ҢеӨ§еӨҡж•°зҡ„йҳ»еЎһж“ҚдҪңгҖӮпјҲзә§еҲ«еӨӘй«ҳдәҶпјҢдёҚиғҪеҲҮжҚўзәҝзЁӢеҺ»жӢҝпјҢдҫӢеҰӮж–Үд»¶пјҢиҰҒи°ғзЎ¬д»¶зҡ„й©ұеҠЁпјҢдҪҶжҳҜзәҝзЁӢеҲҮдёҚиҝҮеҺ»дәҶпјүгҖӮеңЁDISPATCH_LEVELпјҢеҶ…ж ёд»Јз ҒдёҚеә”жү§иЎҢдјҡеҜјиҮҙзәҝзЁӢйҳ»еЎһжҲ–йңҖиҰҒиҫғй•ҝж—¶й—ҙе®ҢжҲҗзҡ„ж“ҚдҪңгҖӮ зә§еҲ«дёә1зҡ„APC_LEVEL дёҚиғҪж“ҚдҪңж–Үд»¶

еңЁMSDNжҹҘAPIд№ҹдјҡжңүжҳҫзӨәпјҢйңҖиҰҒж»Ўи¶ізҡ„IRQL

зә§еҲ«дёҚеҜ№жүҫBugжңүзӮ№йҡҫеҸ—пјҢжҜ•з«ҹдёҚжҳҜеҸӮж•°й—®йўҳпјҢдёҚжҳҜжқғйҷҗй—®йўҳ

жүҖд»ҘеҰӮдҪ•зҹҘйҒ“иҮӘе·ұзҡ„д»Јз ҒеңЁд»Җд№Ҳзә§еҲ«пјҹз”ЁдёҖдёӘAPI KeGetCurrentIrql();

жҸҗеҚҮпјҢйҷҚдҪҺзӯүзә§

дёҖиҲ¬еңЁеҶҷй©ұеҠЁзЁӢеәҸзҡ„ж—¶еҖҷпјҢеңЁзЁӢеәҸе…ҘеҸЈдјҡеҶҷдёҖдёӘ

PAGED_CODE() жҳҜдёҖдёӘеҶ…ж ёе®ҸпјҢз”ЁдәҺзЎ®дҝқжҹҗж®өд»Јз ҒеҸӘеңЁ IRQL дёә PASSIVE_LEVEL ж—¶иҝҗиЎҢгҖӮ жҳҜдёҖдёӘж–ӯиЁҖе®ҸпјҢз”ЁжқҘжЈҖжөӢзә§еҲ«еҜ№дёҚеҜ№

еҶ…ж ёзЁӢеәҸдёӯпјҢеҶҷдёӢиҝҷдёӘд»Јз Ғе°ұдјҡи“қеұҸпјҡ

еӣ дёәжҸҗеҚҮеҲ°DISPATCH_LEVELиҝҷдёӘзә§еҲ«пјҢе°ұе’ҢI/Oз®ЎзҗҶеҷЁдёҖдёӘзә§еҲ«дәҶпјҢиҝ”еӣһзҡ„дёҚиғҪиў«еӨ„зҗҶ

жҸҗеҚҮеҲ°й«ҳ IRQL еҗҺз«ӢеҚіиҝ”еӣһдјҡеҜјиҮҙзі»з»ҹеңЁй«ҳ IRQL дёҠиҝҗиЎҢдёҚе…Ғи®ёеҲҶйЎөжҲ–зӯүеҫ…зҡ„д»Јз ҒпјҢд»ҺиҖҢеј•еҸ‘и“қеұҸгҖӮ
